# ✏️ 自訂模式步驟序列

> 此檔案為自訂模式的權威步驟定義。由 SKILL.md 核心派發載入。

依使用者選擇的完整性等級：

## 🔴 低（Lean）— 共 4 步

```
S1. JTBD 陳述 → 讀取 references/02b-jtbd.md
S2. HMW 一個 → 讀取 references/03-define.md → 2.3
S3. PR-FAQ → 讀取 references/04a-prfaq.md
S4. North Star → 讀取 references/05a-northstar-aha.md
（任一步驟可由使用者替換為其他框架）
────
最終產出 → 產品規格摘要（未執行欄位標記「未執行」）
```

## Reference 載入規則

進入各步驟時，僅讀取對應的 reference 檔案（不要預先全部載入）。各步驟旁已標注對應的 reference 路徑。

## 最終產出格式

**產品規格摘要**（僅整合已完成的步驟，未執行的欄位標記為「未執行」）

完成後，依 `references/rules-end-of-flow.md` 執行流程結束規則。
