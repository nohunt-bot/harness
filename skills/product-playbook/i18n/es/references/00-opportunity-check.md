# Pre-Paso: Evaluación de Oportunidad

> **Nota: Qué pasos ejecutar por modo está definido autoritativamente por las "Secuencias de Pasos por Modo" en SKILL.md. Lo siguiente es solo de referencia.**

**Aplicable: Modo completo, alta completitud, audiencia es ejecutivos/liderazgo**
**Omitible: Modo rápido, modo build directo, audiencia es ingenieros/diseñadores**

Si el usuario está construyendo un producto 0-a-1 desde cero, recorre estas cinco preguntas primero. Un "no" en cualquier pregunta es una señal para replantear:

```
| # | Pregunta de Evaluación | Respuesta del Usuario | Evaluación |
|---|------------------------|----------------------|------------|
| 1 | ¿Esto resuelve un punto de dolor real y urgente del usuario? ¿Quiénes son los primeros clientes beneficiados? ¿Cómo los encontrarás? | | ✅/⚠️ |
| 2 | ¿Tienes una ventaja única para resolver este problema? ¿Los clientes objetivo lo usarán al menos semanalmente? ¿El mercado es lo suficientemente grande? | | ✅/⚠️ |
| 3 | ¿Con los recursos actuales, puedes construir un producto usable en 2-3 años? | | ✅/⚠️ |
| 4 | ¿Cómo es el panorama competitivo? ¿Puedes ganar? ¿Cuál es tu diferenciación? | | ✅/⚠️ |
| 5 | ¿Existe un camino sostenible para el crecimiento de usuarios y la monetización? | | ✅/⚠️ |
```

## Verificación Rápida DHM (Gibson Biddle / Netflix)

¿Puede esta oportunidad lograr:
- **D (Delight)**: ¿Sorprender y superar las expectativas del usuario?
- **H (Hard to copy)**: ¿Ser difícil de replicar para los competidores?
- **M (Margin-enhancing)**: ¿Mejorar los márgenes a medida que crece la escala?

Si algunas señales de mercado no están claras, busca señales direccionales:
- Tendencias macro (p.ej., adopción de IA creando oportunidades de reemplazo de flujos de trabajo)
- Cambios de comportamiento (usuarios ya usando soluciones alternativas, indicando demanda real)
- Mercados análogos (encontrar escenarios comparables validados)

## Verificación de Pasión del Equipo

Confirma que el equipo tiene pasión genuina por este espacio de problemas. Equipos que carecen de motivación intrínseca inevitablemente fallarán en el camino hacia PMF.
