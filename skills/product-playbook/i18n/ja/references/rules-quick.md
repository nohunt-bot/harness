# 🚀 クイックモード ステップ順序（3ステップ + 最終出力）

> このファイルはクイックモードの正式なステップ定義です。SKILL.mdコアディスパッチャーにより読み込まれます。

## ステップ順序

```
S1. JTBD Statement → references/02b-jtbd.md を読み込み
S2. PR-FAQ → references/04a-prfaq.md を読み込み
S3. North Star Metric + Aha Moment → references/05a-northstar-aha.md を読み込み
────
最終出力 → 1ページの方向性サマリー
```

## リファレンス読み込み指示

| ステップ | リファレンスファイル |
|------|---------------|

## 最終出力フォーマット

**1ページの方向性サマリー**：問題 → ソリューション → 成功の定義、の3部構成。クイックブリーフィングやチームシンクに適しています。

完了後、`references/rules-end-of-flow.md`に従ってエンドオブフロールールを実行してください。クイックモードではプレゼンテーションが必要かのみ確認 — PRDや開発ハンドオフパッケージは提案しません（内容が十分包括的ではないため）。
