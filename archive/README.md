# Archive

Resolved history — nothing here is loaded by any session or referenced as a
live rule. Archive-only policy: files move here, they are never deleted.

- `proposals/` — proposals that were approved AND applied (or rejected).
  Open proposals live in `docs/proposals/`; once resolved they move here
  with their status line updated (lifecycle rule: MAINTENANCE.md).

Historical narrative that live rules still cite as rationale
(`LETTER.md`, `docs/DIAGNOSIS.md`, `docs/tasks/*`, `docs/decisions/*`)
stays in place: it costs zero load tokens and moving it would churn
referencing files, including approval-gated ones.
